1. all_idx2type.pckl

2. diag_idx2desc.pckl
- 0~499까지의 한 숫자를 넣으면 해당하는 영문 진료행위를 텍스트로 출력

3. diag_idx2desc(kor).pckl
- 0~499까지의 한 숫자를 넣으면 해당하는 영문 진료행위를 텍스트로 출력
- 영문은 500개 중 50여 칸이 빠져있는데, 한글은 몇 칸 안 빠져있습니다.

4. sick_idx2desc.pckl
- 500~767까지의 상병명 텍스트에 해당

5. pres_idx2desc.pckl
- 768~1399까지의 처방 텍스트에 해당

[Outputs]

1. out_idx2desc.pckl
- 0~499까지의 상병명 텍스트에 해당
